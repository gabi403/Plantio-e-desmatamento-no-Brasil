# 🌱 Agro Forte, Futuro Sustentável

projeto criado para o projeto Agrinho 2026

Uma página web educativa e responsiva que apresenta informações sobre sustentabilidade no agronegócio brasileiro, técnicas agrícolas sustentáveis, preservação ambiental e dados sobre os principais biomas do Brasil.

## 📋 Sobre o Projeto

O projeto *Agro Forte, Futuro Sustentável* foi desenvolvido para conscientizar sobre a importância do equilíbrio entre a produção agrícola e a preservação do meio ambiente.

A aplicação apresenta informações de forma visual e organizada, utilizando um design moderno e intuitivo para facilitar a compreensão dos conteúdos relacionados ao agronegócio sustentável.

## ✨ Funcionalidades

* Página totalmente responsiva;
* Navegação suave entre seções;
* Menu fixo durante a rolagem;
* Layout moderno com foco em acessibilidade visual;
* Cards informativos com efeitos de interação;
* Estatísticas sobre o agronegócio brasileiro;
* Técnicas sustentáveis para produção agrícola;
* Dicas para melhorar o plantio;
* Informações sobre desmatamento nos biomas brasileiros;
* Dados sobre Mata Atlântica, Cerrado e Amazônia.

## 🛠️ Tecnologias Utilizadas

* *HTML5*
* *CSS3*
* *JavaScript*
* *Google Fonts (Poppins)*

## 📊 Conteúdo da Página

### 🌱 Introdução

Apresenta a importância da agricultura sustentável para o desenvolvimento econômico e ambiental.

### 📈 Dados e Estatísticas

Indicadores relevantes sobre o agronegócio brasileiro:

* Participação no PIB nacional;
* Produção de grãos;
* Uso da água na agricultura;
* Preservação da vegetação nativa.

### 🚜 Técnicas Sustentáveis

Práticas modernas que contribuem para uma produção mais eficiente e sustentável:

* Plantio Direto;
* Irrigação Inteligente;
* Energia Solar;
* Integração Lavoura-Pecuária-Floresta (ILPF);
* Agricultura de Precisão;
* Compostagem.

### 🌾 Dicas para Melhorar o Plantio

* Adubação Orgânica;
* Análise do Solo;
* Irrigação por Gotejamento;
* Rotação de Culturas.

### ⚠️ Desmatamento nos Biomas Brasileiros

Informações sobre os desafios ambientais enfrentados por:

* Mata Atlântica;
* Cerrado;
* Amazônia.

### 🌳 Biomas Brasileiros

Apresenta dados históricos e atuais relacionados à conservação ambiental dos principais biomas do país.

## 🎨 Características Visuais

* Tema inspirado na natureza;
* Paleta de cores verdes e neutras;
* Cabeçalho com imagem de fundo em tela cheia;
* Efeitos de hover nos botões e cartões;
* Layout organizado em grids responsivos.
